# zoom-expert

Senior Zoom Apps Architect and Marketplace Consultant.

## Installation

### Local (Current Workspace)
This skill is already available in the `.gemini/skills/zoom-expert/` directory.

### Global
To install this skill globally (for all projects):
1. Copy the `.gemini/skills/zoom-expert/` folder to your global agents directory (e.g., `C:\Users\THIAGO\.agents\skills\`).
2. Refresh your agent session.

## Usage

Trigger this skill by asking for:
- "Implement Zoom App features"
- "Zoom Apps SDK help"
- "Publish Zoom App Marketplace"
- "Handle Zoom events"
- "SPH Timer implementation details"

## Key Features

- **End-to-End lifecycle support:** From architecture to submission.
- **Project Context aware:** Pre-configured with SPH Timer patterns.
- **Security-First:** Focused on Privacy-by-Design and secure OAuth flows.
- **Tooling:** Integrated guidance for `zoomSdk`, REST APIs, and Webhooks.

## Directory Structure

- `SKILL.md`: Core agent instructions.
- `references/`: Detailed technical guides.
- `examples/`: Code templates and sample implementations.
- `scripts/`: Maintenance and validation utilities.
